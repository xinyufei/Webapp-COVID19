# Webapp-COVID19
Website app for operations research problems in COVID-19  
Based on https://github.com/Huiwen-git/Web-app

To run this app, after downloading it, you can run

    .. code-block:: bash

    SET FLASK_APP=application.py
    flask run
    
in cmd or 

    .. code-block:: bash

    SET FLASK_APP=application.py
    flask run

in Bash.
